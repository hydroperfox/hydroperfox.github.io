--- title: Whack™ engine layout: default ---

The Whack™ engine will allow client side and server side interactive applications to be built using ActionScript 3, targeting HTML5 and Node.js®.

Once ActionScript 3 support lands, two additional languages, MXML and Cascading Style Sheet (CSS), will be in the implementation road, which will be a 1-1 mapping to ActionScript 3.

Note that Whack™ is not a Flash Player implementation; it is a brand new platform with many fundamental differences including the API and project setup.

Whack SDK

The Whack SDK contains a tool that is both a package manager and a build tool incorporated into one.

ActionCore

ActionCore is an ActionScript 3 runtime implemented in JavaScript, which evaluates in HTML5 and Node.js® runtimes.

Whack libraries

The Whack libraries repository contains Whack packages that implement ActionScript 3 built-ins.
